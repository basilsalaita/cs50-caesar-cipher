# Caesar Cipher Encryption Tool (C)

This project implements the Caesar cipher encryption algorithm in C.

The program takes a plaintext message and a numeric key from the user, then shifts each letter by the key value to produce encrypted text.

## Features
- Command-line encryption
- Input validation for numeric keys
- Handles uppercase and lowercase letters

## Skills
- C Programming
- Algorithms
- String manipulation
- Command-line programs
